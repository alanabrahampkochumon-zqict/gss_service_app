# CS320 Project

This project was done as a part of CS320: Software Automation and Testing by Southern New Hampshire University.
The end goal was to create a 3 specific features, contact, appointment, and Task services and its enclosing (data) class and writing unit testing on them.
Build System: Gradle with Kotlin
Run Environment: Windows 11 with Intellij Idea 2025.3.1
Code Coverage: 100% (98% in branching logic)

## How to run

This is an intellij idea project. You can run the project as it using intellij idea or use

```bash
    gradlew test
```

## Keytakeaways

The goal of the course was to ensure that students are equipped with testing phase of SDLC.

NOTE: This project doesn't contain a main class or function. It was created purely for testing.
